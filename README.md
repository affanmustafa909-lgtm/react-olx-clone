# React OLX Clone

A simple OLX-style marketplace web application built using React.
Users can browse product listings and view item details in a clean and responsive interface.

---

## 🚀 Features

* View product listings
* Simple and clean user interface
* Component-based architecture using React
* Responsive layout for different screen sizes

---

## 🛠️ Tech Stack

* React
* CSS

---



---

## ⚙️ Installation

Clone the repository and install dependencies.

```
git clone https://github.com/your-username/react-olx-clone.git
cd react-olx-clone
npm install
```

---

## ▶️ Run the Project

Start the development server.

```
npm start
```

The app will run at:

```
http://localhost:3000
```
